# ScheduGraph 📊
ScheduGraph is where you can save your stuff, links and other things you need to do. You can come back to it to check your things, finish it and mark them off the list. You can also save notes, and other pages you need to go to later on.
## Notes
In notes, you can type things you need to save so you can come back later on to see it ✔️.<br>
You can also use notes to save links, URLs and keys. All your stuff are safe and no one else can see it 🔒.

## Schedule
You keep your schedules in a graph like place. It holds simple things like events to complex ones like conference meetings.<br>
When you are done with a meeting or event, you can simply click the DONE button to remove it when you are done.<br>
It saves them like a to-do list app 📖.<br>
